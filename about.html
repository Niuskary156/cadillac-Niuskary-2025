About This Project

My name is Niuskary, and I created this site as part of my web development course. I chose Cadillac’s electric cars because they represent luxury and sustainability, values that inspire me. I customized the design with violet accents because it’s my favorite color and part of my identity.

This project helped me practice HTML, CSS, and JavaScript while building a complete multi-page website. It is not an official Cadillac site, but a student project that reflects my creativity and technical skills.
